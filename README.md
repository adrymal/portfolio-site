# Hugo Portfolio

Портфолио на Hugo + Tailwind CSS.

## Локальный запуск

```bash
npm install
npm run dev
```

Сайт будет доступен на `http://localhost:1313/` или на следующем свободном порту.

## Продакшен-сборка

```bash
hugo
```

Итоговая папка сборки: `public/`.

## Cloudflare Pages

- Build command: `hugo`
- Build output directory: `public`
- Framework preset: `Hugo`
